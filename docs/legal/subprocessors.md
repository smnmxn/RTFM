# Sub-processors

**Last updated:** January 30, 2025

---

supportpages.io uses the following third-party sub-processors to provide our services. These companies process data on our behalf under contractual obligations that protect your data.

---

## Current Sub-processors

| Company | Purpose | Location | Data Processed |
|---------|---------|----------|----------------|
| **Anthropic** | AI analysis and documentation generation | USA | Code diffs, project context, generated content |
| **GitHub** | Authentication, repository access, webhooks | USA | User identity, repository metadata, code diffs |
| **Hetzner** | Cloud infrastructure and hosting | Finland (EU) | All stored data |
| **Cloudflare** | DNS, custom domains, SSL certificates | Global | Domain names, traffic routing |
| **Rollbar** | Error monitoring and alerting | USA | Error logs (sensitive data filtered) |

---

## Sub-processor Details

### Anthropic
**Website:** [anthropic.com](https://www.anthropic.com)
**Privacy Policy:** [anthropic.com/privacy](https://www.anthropic.com/privacy)

Anthropic provides the Claude AI that powers our documentation generation. When you analyze code:
- Code diffs and context are sent to Anthropic's API
- Anthropic processes the data to generate documentation
- Data is not retained after processing
- Data is not used to train AI models

**Data Processing Agreement:** Yes
**Transfer Mechanism:** Standard Contractual Clauses

---

### GitHub
**Website:** [github.com](https://github.com)
**Privacy Policy:** [docs.github.com/privacy](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)

GitHub provides authentication and repository access:
- OAuth authentication for user sign-in
- GitHub App for repository integration
- Webhooks for PR/commit notifications

**Data Processing Agreement:** GitHub DPA applies
**Transfer Mechanism:** EU-US Data Privacy Framework, Standard Contractual Clauses

---

### Hetzner
**Website:** [hetzner.com](https://www.hetzner.com)
**Privacy Policy:** [hetzner.com/privacy](https://www.hetzner.com/legal/privacy-policy)

Hetzner hosts our infrastructure in Finland:
- Database storage
- Application servers
- File storage

**Location:** Finland (within EU)
**Data Processing Agreement:** Yes
**GDPR Compliant:** Yes (EU-based)

---

### Cloudflare
**Website:** [cloudflare.com](https://www.cloudflare.com)
**Privacy Policy:** [cloudflare.com/privacy](https://www.cloudflare.com/privacypolicy/)

Cloudflare provides DNS and custom domain functionality:
- DNS management
- Custom hostname SSL certificates
- Traffic routing

**Data Processing Agreement:** Yes
**Transfer Mechanism:** Standard Contractual Clauses, EU representative designated

---

### Rollbar
**Website:** [rollbar.com](https://rollbar.com)
**Privacy Policy:** [rollbar.com/privacy](https://rollbar.com/privacy/)

Rollbar monitors errors in our application:
- Error tracking and alerting
- Stack traces and debugging info
- **Note:** We filter sensitive data (tokens, code) from error logs

**Data Processing Agreement:** Yes
**Transfer Mechanism:** Standard Contractual Clauses

## Changes to Sub-processors

We'll notify you before adding or changing sub-processors:

1. **Update this page** at least 30 days before the change
2. **Email notification** (if you've opted in to legal updates)
3. **Right to object** within 30 days of notification

If you have concerns about a sub-processor change, contact us at **dpa@supportpages.io**.
