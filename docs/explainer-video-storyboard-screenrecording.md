# SupportPages Explainer Video Storyboard

**Duration: 90 seconds | 12 scenes**

## Pre-Production Setup

### Demo Environment
- [ ] SupportPages instance with sample project "Acme App"
- [ ] GitHub repo with realistic PR history
- [ ] Pre-populated help center with 6-8 articles
- [ ] AI chat with trained responses
- [ ] Clean browser (no bookmarks bar, minimal UI)
- [ ] Dark mode enabled across all interfaces

### Recording Setup
- **Resolution:** 2560x1440 or 3840x2160 (scale down in edit)
- **Frame rate:** 60fps (allows smooth slow-motion)
- **Tool:** OBS, ScreenFlow, or Loom
- **Cursor:** Enlarged, smooth movement, highlight clicks
- **Browser:** Arc or Chrome with minimal chrome

### Screen Recording Tips
- Move cursor slowly and deliberately
- Pause briefly before each click
- Use keyboard shortcuts where possible for snappier UI
- Record each scene 2-3x for best take
- Leave 2 seconds of buffer at start/end of each clip
